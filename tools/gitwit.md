## What Is Collected

- Page views and usage analytics (via Vercel Analytics)
- App descriptions and prompts
- Generated code (processed server-side)
- Session and visitor metadata
- Performance metrics

## What Is NOT Collected

- Vercel Analytics is privacy-focused (no cookies, no PII)
- Specific exclusions not documented beyond Vercel Analytics defaults

## How to Fully Opt Out

No documented opt-out mechanism for telemetry.

1. **Self-host:** Clone the repository and run locally to avoid Vercel Analytics
2. **Core processing:** Server-side code generation cannot be opted out of in hosted mode

## Red Flags

- Vercel Analytics tracks usage on the hosted platform
- All code generation happens server-side
- No documented opt-out
- App descriptions and prompts processed on GitWit servers
