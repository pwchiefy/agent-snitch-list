## What Is Collected

- Design files uploaded for conversion (Figma imports)
- Generated code (processed server-side)
- Usage analytics via cookies
- Account and session data
- Project metadata

## What Is NOT Collected

- Not documented in detail

## How to Fully Opt Out

1. **Cookies:** Manage via cookie consent banner
2. **Core processing:** Cannot opt out of server-side processing (required for functionality)
3. **Account deletion:** Request via account settings

## Red Flags

- Design files are uploaded to Codejet servers
- All code generation is server-side
- Closed source with limited transparency
- Cookie-based tracking on web platform

## Evidence

- Website: https://www.codejet.ai/
- Cookie consent mechanism on website
- Server-side architecture for design-to-code conversion
