## What Is Collected

- Design files uploaded for conversion (Figma designs)
- Generated code (processed on Kombai servers)
- Usage analytics
- Account and session data
- Project metadata and design structure

## What Is NOT Collected

- Not documented in detail

## How to Fully Opt Out

There is no documented way to opt out of data collection while using the service.

1. **Only option:** Stop using the service and request account deletion

## Red Flags

- No documented opt-out for telemetry or data collection
- Design files processed on Kombai servers
- All code generation is server-side
- Closed source with limited transparency
- No published security certifications
