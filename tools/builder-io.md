## What Is Collected

- Design files (Figma) processed for code generation
- Generated code and conversion metadata
- Usage analytics
- Account and organization data
- Editor/plugin interaction patterns

## What Is NOT Collected

- **Privacy Mode:** When enabled, reduces data collection and processing
- Specific exclusions depend on Privacy Mode configuration

## How to Fully Opt Out

1. **Privacy Mode:** Enable in Builder.io settings to reduce data collection
2. **Plugin removal:** Uninstall the Figma plugin to stop design file processing
3. **Account deletion:** Request via account settings

## Red Flags

- Design files sent to Builder.io servers for processing
- Privacy Mode existence implies non-Privacy Mode collects more
- Exact scope of Privacy Mode restrictions not fully documented
- Platform analytics ON by default
