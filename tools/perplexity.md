## What Is Collected

- All search queries and chat prompts
- Conversation history (retained by default)
- Usage analytics (via Google Analytics and custom)
- Browser/device metadata
- Account and session data
- IP address and approximate location
- Uploaded files and images

## What Is NOT Collected

- Perplexity states it does not sell personal data
- Specific exclusions vary by plan

## How to Fully Opt Out

1. **AI data retention:** Account Settings > toggle off conversation history retention (if available on your plan)
2. **Cookies/GA:** Use browser privacy tools to block Google Analytics
3. **Pro plan:** May offer additional data controls
4. **Full opt-out:** Not possible while using the service -- queries must be processed server-side

## Red Flags

- Google Analytics means behavioral data flows to Google
- All queries stored on Perplexity servers by default
- AI data retention is ON by default
- Code included in queries is processed and potentially retained
- Closed source -- cannot verify data handling
- Free tier users have fewer privacy controls
